# vpn-checker

Веб-интерфейс для Raspberry Pi (Bookworm): вставляете ссылку на подписку с
кучей VPN, приложение проходит по всем узлам, проверяет каждый через
`google.com` и для рабочих замеряет скорость. Результаты показываются в живой
таблице с сортировкой по скорости.

Поддерживаемые протоколы: **VLESS** (tcp/ws/grpc/http, tls/reality),
**VMess**, **Trojan**, **Shadowsocks**.

## Как это работает

1. Подписка скачивается и декодируется (base64-блок или готовый список ссылок).
2. Каждая ссылка превращается в outbound-конфиг Xray.
3. Для узла поднимается локальный Xray с SOCKS-входом на `127.0.0.1:10810`.
4. `curl --socks5-hostname` запрашивает `google.com/generate_204`:
   ответ `204` → узел жив, заодно меряется пинг.
5. Для живого узла качается тестовый файл (Cloudflare, 20 МБ) → скорость в Мбит/с.
6. Xray останавливается, переходим к следующему узлу.

Узлы проверяются **по очереди** — иначе параллельные загрузки делят канал и
скорость получается заниженной.

## Установка

```sh
git clone <repo> vpn-checker
cd vpn-checker
chmod +x install.sh
./install.sh
```

Скрипт поставит Xray-core, создаст `.venv` с зависимостями и предложит
настроить автозапуск через systemd.

## Запуск

Вручную:

```sh
.venv/bin/python app.py
```

Откройте `http://<ip-малинки>:3333`, вставьте ссылку подписки, нажмите
**Проверить**.

Если ставили systemd-сервис — он уже слушает порт 3333:

```sh
sudo systemctl status vpn-checker
sudo journalctl -u vpn-checker -f
```

## Настройка (переменные окружения)

| Переменная | По умолчанию | Назначение |
| --- | --- | --- |
| `XRAY_BIN` | `/usr/local/bin/xray` | путь к бинарю Xray |
| `VC_SOCKS_PORT` | `10810` | локальный порт SOCKS для проверки |
| `VC_CONNECT_URL` | `https://www.google.com/generate_204` | чем проверяется доступность |
| `VC_CONNECT_TIMEOUT` | `8` | таймаут проверки доступности, с |
| `VC_SPEED_URL` | `https://speed.cloudflare.com/__down?bytes=20000000` | тестовый файл для скорости |
| `VC_SPEED_TIMEOUT` | `15` | таймаут замера скорости, с |

Пример: проверять YouTube вместо google и качать 50 МБ —

```sh
VC_CONNECT_URL=https://www.youtube.com/generate_204 \
VC_SPEED_URL='https://speed.cloudflare.com/__down?bytes=50000000' \
.venv/bin/python app.py
```

## Структура

```
app.py             Flask: страница + SSE-поток результатов
parser.py          разбор подписки и ссылок в outbound-конфиги Xray
checker.py         запуск Xray и тесты (доступность + скорость)
templates/index.html  веб-интерфейс
install.sh         установка Xray, venv, systemd
```

## Замечания

- Проверка занимает несколько секунд на узел (доступность + загрузка 20 МБ).
  Для 50+ узлов это минуты — результаты появляются по мере готовности.
- Используйте только со своими подписками.
